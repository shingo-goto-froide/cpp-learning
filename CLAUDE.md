# C++ 学習プロジェクト

## 概要
「江添亮のC++入門」に沿ってC++を学習中のプロジェクト。

## 学習者の背景
- C# 実務経験4年
- C++との違いを対比して説明すると理解しやすい

## ノート作成ルール
- 各章の学習後、`notes/` フォルダに要点・補足情報をまとめる
- ファイル名: `chXX_章タイトル.md`（例: `ch03_変数と初期化.md`）
- 内容構成:
  - **要点**: 章の重要ポイントをコード例付きで簡潔にまとめる
  - **補足情報**: 書籍に書かれていない実務的な知識やC#との違いなど
- C#経験者の視点で「C++ではここが違う」というポイントを積極的に記載する
- 既存ノートのスタイル・構成に合わせる

## 開発環境
- Windows 11 / MSYS2 (g++)
- C++23 (`-std=c++23`)
- コンパイルオプション: `-Wall -pedantic-errors`
- ビルド: `mingw32-make run` で実行（VSCodeでは `Ctrl+Shift+B`）
- プリコンパイル済みヘッダー `all.hpp` を使用（標準ヘッダーをまとめてinclude）

## 学習進捗の自動管理
- `notes/` 内の最新ファイルから現在の進捗を自動判定する
- 最新ノートが `ch06` なら、現在は第7章を学習中と判断する
- ユーザーが「終わり」「次」等と言ったら章の切り替えと判断する
- ユーザーに章番号を聞かない。わからなければ `notes/` を確認する

## 返答ルール
- 日本語で回答する
- 簡潔・明確に書く
